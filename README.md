# Pokédex API

API REST para gestionar una Pokédex personal de usuarios, desarrollada
como práctica del Módulo 2 (APIs REST e Integración) de Sistemas de Información (UFV).

## Stack

- FastAPI
- SQLModel + SQLite
- JWT (python-jose)
- slowapi (rate limiting)
- httpx (cliente HTTP a PokeAPI)
- reportlab (generación de PDFs)

## Instalación

```bash
uv venv .venv
uv sync
